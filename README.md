# Dr. Audit - AI Resume Health Checker

An intelligent Applicant Tracking System (ATS) calculator. It evaluates Resumes against Job Descriptions using a Hybrid AI Architecture (Python Fast API + Node.js Telegram Bot + LLM Extraction & Math).

## Services

- `/bot`: Node.js Express/Telegraf server for the UI
- `/ai`: Python Fast API service for document parsing, string matching, and LLM communication.
